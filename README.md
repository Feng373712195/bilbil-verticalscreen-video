##  如何添加到Chrome

- 先把项目拉到本地
`git clone git@github.com:Feng373712195/bilbil-verticalscreen-video.git`

- Chomre打开`chrome://extensions/`

- 打开开发者模式

- 选择加载已解压的扩展程序，选择插件文件夹

- 添加成功，记得开启
